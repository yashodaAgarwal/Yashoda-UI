Introduction

Yashoda-UI is a continuously developing Component Library, built using HTML5 and CSS3 only. You can use it to build awesome UI layouts. To know more about it, look at its GitHub repo.

Quick Start

Copy and paste the code in the head tag of your html document. So that you can style your components by just adding class names to your html elements.

CSS

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.
